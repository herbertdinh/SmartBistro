# SmartBistro

Agile-Based Restaurant Management System prototype for SENG205 Assessment 3.

SmartBistro is a web-based restaurant management prototype designed to support QR ordering, kitchen display coordination, inventory tracking, reservation management, and basic admin reporting.

## Core Features

- QR-based digital menu
- Cart and checkout workflow
- Kitchen Display System (KDS) order queue
- Reservation management
- Inventory tracking and low-stock alerts
- Admin dashboard for sales, orders, and stock overview

## Technology Stack

| Layer | Technology |
|---|---|
| Front-end | HTML, CSS, JavaScript |
| Back-end prototype | PHP |
| Database | MySQL |
